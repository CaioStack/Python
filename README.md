## 💡 Como Estudar

1. **Leia o código** completo antes de executar
2. **Execute o arquivo** e observe a saída: `python variaveis.py`
3. **Experimente modificar** os exemplos para ver o que acontece
4. **Tente os exercícios** ao final de cada arquivo sem olhar a solução
5. Só avance para o próximo tema quando se sentir confortável

---

## ⚡ Como Executar

```bash
# Exemplo — rode qualquer arquivo diretamente:
python 01_variaveis_e_tipos/variaveis.py
python 11_poo/poo.py
```

Requisito: Python 3.10+ (para usar match/case do módulo 03)
